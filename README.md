Proyek Klasifikasi Citra Daun ini merupakan studi mendalam dalam bidang Pengolahan Citra Digital (DIP) dan Machine Learning. Tujuan utamanya adalah mengidentifikasi jenis-jenis daun secara otomatis. Kode proyek ini mencakup seluruh alur kerja: mulai dari Pra-pemrosesan Citra untuk membersihkan gambar, Ekstraksi Fitur untuk mengambil karakteristik visual daun (seperti warna, bentuk, dan tekstur), hingga proses Klasifikasi menggunakan algoritma K-Nearest Neighbors (KNN). Semua eksperimen dan langkah image processing didokumentasikan dalam Jupyter Notebook di folder notebooks/, sementara kode utama untuk menjalankan klasifikasi berada di folder src/. Proyek ini menggunakan bahasa Python dan pustaka populer seperti OpenCV dan Scikit-learn. Untuk menjalankan proyek, Anda cukup menginstal dependensi yang diperlukan dan menjalankan skrip main.py di folder src/.
